# Sistema-IOT-para-Monitoramento-de-Press-o-e-Vaz-o-ESP32
Sistema IOT para simular o monitoramento de pressão e vazão utilizando um ESP32 e a plataforma ThingSpeak para acesso remoto

Este código foi projetado para o ESP32 DevKit Wroom-32.

Para utiliza-lo você deve instalar a biblioteca do Esp32 "Arduino_ESP32_OTA" e a "Liquid Crystal"

O código foi feito para simular os sensores de vazão e pressão YF-S201 e MPX-5010 com a entrada de um potenciômetro de 10k
